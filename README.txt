# database-application

W folderze screeny, możemy zobaczyć screeny działania programu.

Program do obsługi bazy danych MySQL. Program działa w konsoli. Umożliwia tworzenie/usuwanie tabel, kolumn, rekordów. Wyświetlanie określonych tabel, kolumn, rekordów. Wykorzystano m.in. biblioteki: mysql.connector, io, re, prettytable, colorama. Należy te biblioteki mieć zainstalowane podczas uruchomienia programu.
 
Aby uruchomić program, w pierwszej kolejności musimy utworzyć bazę danych MySQL na serwerze. Następnie przechodzimy do pliku db_connect.py i wpisujemy dane do logowania się z bazę danych.
Następnie możemy uruchomić program. Należy plik main.py uruchomić w interpreterze Pythona. Kolejne kroki dokonujemy według wyświetlanej w terminalu instrukcji.
